# Sprint Planner

A clean, intuitive tool for agile teams to plan sprints by calculating team capacity and estimating commitment.

## Live Demo
Try it here: [https://bhumiksoni.github.io/sprint-planner/](https://bhumiksoni.github.io/sprint-planner/)

## What It Does

Add team members, track their days off, and instantly see how many story points your team can commit to in the upcoming sprint.

## Key Features

- **Simple Team Management**: Just add names - no complex setup
- **Quick Leaves Tracking**: Set days off with a single click
- **Real-time Calculations**: See sprint capacity update instantly
- **Clean Interface**: Split-screen design - team on left, results on right
- **No Data Storage**: Everything runs in your browser

## How It Works

1. **Add Team Members** → Just enter names
2. **Set Days Off** → For each team member
3. **Adjust Settings** → Sprint days and commitment percentage
4. **Get Results** → See recommended sprint points

## Calculations

- Each team member contributes 1 point per working day
- Working days = Sprint days - Days off
- Available points = Sum of all team members' working days
- Recommended commitment = Available points × Commitment %
